### Naming a User
Usernames should contain only lowercase letters, numbers, and underscroes. They should NOT contain spaces, dashes, or any other character.

**Good Usernames**
- appacademy
- patel_kush_12
- bdorsey

**Bad/Incorrect Usernames**
- Ned Ruggeri
- melvin-howard-torme
- b.d.o.r.s.e.y
